How to use this tool - IMPORTANT

  1. Create a bot via @BotFather and get your bot token
  2. Start your bot by sending "/start" message to it
  3. Get your Chat ID by:
    • Visit: https://api.telegram.org/botYOUR_TOKEN/getUpdates
    • OR click "Get Chat ID Helper" button above
    • Look for "chat":{"id":123456789} in the response
  4. Test connection with the "Test Connection" button
  5. Upload files - Use "Alternative Method" if regular method fails

Note: Telegram limits: Max 50 MB per file. For frontend, recommended max is 20 MB.
